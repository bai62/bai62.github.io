NumPy之ndarray创建，属性与轴

PyTorch 深度学习
AI
发布日期:   2021-10-20

ndarray特点

  • ndarray(N-dimensional-array)
  • numpy.ndarray
  • 不同于python列表，ndarray长度不可变
  • 相同的操作，使用更少的内存，速度更快

创建ndarray数组

  • np.array()深拷贝；np.asarray()浅拷贝

  • array和asarray都可以将结构数据转化为ndarray，但是主要区别就是当数据源是ndarray时，array仍然会copy出一个副本，占用新的内存，但asarray不会。

import numpy as np

arr = [[1,1,1],[1,1,1],[1,1,1]]
mat = np.ones((3,3)) # tuple(3,3)
m_arr1 = np.array(arr)
m_asarr1 = np.asarray(arr)

m_arr2 = np.array(mat)
m_asarr2 = np.asarray(mat)

arr[1] = 0
mat[1] = 0

print(m_arr1,"\n\n",m_asarr1,"\n________\n",m_arr2,"\n\n",m_asarr2)

ndarray的属性

ndim

  • N-dimension
mat.ndim #得到numpy数组的维数

shape

  • (B,W,H,C)数据，使用input_data.shape[1:]即可获取到数据的形状
a.shape         (400, 500, 3)
a.shape[1]      500
a.shape[1:]     (500, 3)
a.shape[1::]    (500, 3)
a.shape[:-1]    (400, 500)
a.shape[:]      (400, 500, 3)
a.shape[::]     (400, 500, 3)
a.shape[::-1]   (3, 500, 400)
a.shape[1::-1]  (500, 400)
>>>np.arange(6).reshape(2,3)
>>>array([[0, 1, 2],
>>>       [3, 4, 5]])

size

  • shape属性中元素乘积
  • 二维面积，三维体积
>>> mat.size
6

dtype

  • 查看数组元素类型
>>> np.ones((3,3)).dtype
dtype('float64')
  • ndarray可以在创建时通过dtype指定类型
>>> np.ones((3,3),dtype='float').dtype
dtype('float64')

>>> np.asarray([[1,2],[3,4]],dtype='float32').dtype
dtype('float32')
  • 可以使用 astype() 改变数组的数据类型，但是改变数据类型会创建一个新的数组
>>> a = np.ones((3,3),dtype = 'int32')
>>> b = a.astype('float')
>>> a.dtype
dtype('int32') //原数组的数据类型没有改变
>>> b.dtype
dtype('float64')

通过numpy函数创建ndarray

np.ones() 和 np.zeros()

  • 默认float64
>>> np.ones(shape=(2,3),dtype='int32') * 0.5
array([[0.5, 0.5, 0.5],
       [0.5, 0.5, 0.5]])

>>> np.zeros((3,3))+2
array([[2., 2., 2.],
       [2., 2., 2.],
       [2., 2., 2.]])

np.arange()和np.linspace()

arange()类似于内置函数range()，通过指定开始值、终值和步长创建表示等差数列的一维数组，注意得到的结果数组不包含终值。

linspace()通过指定开始值、终值和元素个数创建表示等差数列的一维数组，可以通过endpoint参数指定是否包含终值，默认值为True，即包含终值。

np.arange([start, ]stop, [step, ]dtype=None)

np.linspace(start, stop, num=50, endpoint=True, retstep=False, dtype=None)

  • num：序列中元素的个数，默认是 50
    • 步长为(stop - start)/num
  • endpoint：默认为 True，如果为 True，则数组最后一个元素是 stop
  • retstep：默认为 False，如果为 True，则返回数组与公差
>>> np.arange(5)
array([0, 1, 2, 3, 4])

>>> np.arange(3,8,2,dtype='int32')
array([3, 5, 7])

数组的轴

  • 当 axis=i 时，第 i 个轴的数据将会被聚合到一起
  • 形状为 (a, b, c) 的数组，沿着 0 轴聚合后，形状变为 (b, c)；沿着 1 轴聚合后，形状变为 (a, c)
>>> mat = np.random.randint(10, size=(4, 3))
>>> mat
array([[2, 5, 3],
       [6, 5, 1],
       [2, 4, 0],
       [5, 1, 4]])
>>> np.sum(mat,axis=0)//将数组按0轴方向聚合↓
array([15, 15,  8])
>>> np.sum(mat,axis=1)//将数组按1轴方向聚合→
array([10, 12,  6, 10])
